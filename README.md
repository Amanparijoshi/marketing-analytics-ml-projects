# marketing-analytics-ml-projects
Marketing analytics projects including engagement clustering and sales prediction using machine learning.
Marketing Analytics & Machine Learning Projects

This repository contains two applied machine learning projects focused on marketing analytics and data-driven decision-making.

The projects demonstrate practical implementation of supervised and unsupervised learning techniques using real-world datasets.

Project 1: Facebook Live Sellers – Engagement Analysis
Objective

To analyze Facebook engagement metrics and identify patterns affecting post performance using clustering techniques.

Techniques Used

Exploratory Data Analysis

Correlation Analysis

K-Means Clustering

Elbow Method

Silhouette Score

Key Insights

Engagement peaks during evening hours (6 PM–8 PM).

Video posts generate higher interaction than photos or links.

Comments strongly influence shares.

Optimal clusters identified: 3

Silhouette Score: 0.659 (good separation)

Business Impact

Optimize posting time.

Shift content strategy toward video.

Use clustering to segment high-performing posts.

Project 2: Sales Prediction Using Advertising Data
Objective

To predict product sales based on advertising expenditure across TV, Radio, and Newspaper channels using regression analysis.

Techniques Used

Correlation Analysis

Multiple Linear Regression

Model Evaluation (R², MSE)

Feature Importance Analysis

Key Findings

TV advertising has the strongest impact on sales.

Radio has moderate influence.

Newspaper has minimal impact.

Model R² ≈ 0.88–0.90 (strong predictive performance).

Removing TV significantly reduces model accuracy.

Business Impact

Optimize marketing budget allocation.

Prioritize high-ROI channels.

Use predictive analytics for sales forecasting.

Technologies Used

Python

Pandas

NumPy

Matplotlib

Seaborn

Scikit-Learn

Overall Conclusion

These projects demonstrate how machine learning can be applied to marketing analytics to improve engagement strategy and optimize advertising investments. The Facebook analysis uses unsupervised learning to segment engagement patterns, while the Sales Prediction project applies supervised learning to forecast revenue outcomes. Together, they highlight the value of data-driven decision-making in business strategy.
